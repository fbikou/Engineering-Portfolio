# PDF自動印刷ツール
「GASによるGMail添付PDFファイル自動選別・アップロードツール」で保存された複数のPDFファイルをまとめて自動で印刷する為のツールです。
## 依頼背景(作成背景)

## 機能
PDFファイルが入っているフォルダを読み込み, そのフォルダに入っているPDFファイルを全て自動で印刷します。尚, 使用されるプリンターはPCで設定している既定のプリンターです。
